# ComfyUI-TY-OrbitSphereLight

`TY Orbit Sphere Light` is a standalone ComfyUI custom node project for clean sphere-based relighting reference images with a TY orbit gizmo workflow.

## Status

This repository page is prepared first so the project has a stable GitHub address.

The final plugin package and example workflow will be uploaded after the node package is fully finalized.

## Planned Node Identity

- Node name: `TY Orbit Sphere Light`
- Class name: `TYOrbitSphereLightNode`
- Category: `TY/Relight`

## Project Direction

This project is intended to:

- keep the final render image clean
- keep control handles visible only inside the node preview
- provide a more intuitive orbit-style control experience for azimuth, elevation, and intensity

## Project Origins

This TY node is based on a custom integration direction that references:

- `Sphere-Light-Render-ComfyUI` as the clean sphere-light output baseline
- `comfyui_gaussian_splat` as the interaction reference for orbit-style handle control
